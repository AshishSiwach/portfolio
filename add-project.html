How to use

  1. Fill in all the fields below and click Generate JSON
  2. Click Copy to copy the generated entry
  3. Open projects.json in your editor
  4. Paste it as a new item inside the [ ] array (add a comma after the previous entry)
  5. Save → push to GitHub → Vercel deploys automatically
